# IT1214-Student-Grade-System

## Project Purpose
This is a Java console-based Student Grade Management System developed for a private educational institute. It allows teachers to manage student academic records digitally instead of using paper or spreadsheets.

## Features
- Add new student records
- Display all students
- Search for a student by ID
- Calculate average marks of all students

## How to Run the Program

### Requirements
- Java JDK installed on your computer

### Steps
1. Clone the repository:
   git clone https://github.com/arabdurrahman2211/-IT1214-Student-Grade-System.git

2. Navigate to the project folder:
   cd IT1214-Student-Grade-System

3. Compile the program:
   javac Main.java

4. Run the program:
   java Main
